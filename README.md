# Anomaly Detection on Credit Card Fraud Dataset

Tugas Data Mining – Kelas Asinkronus  
Deadline: 24 November 2025

Repository ini berisi implementasi tiga pendekatan anomaly detection pada dataset Credit Card Fraud:
1. Statistical Approach (Elliptic Envelope)
2. Proximity Approach (kNN-based Outlier Detection)
3. Density Approach (Local Outlier Factor)

Seluruh eksperimen dilakukan di dalam satu notebook:
`anomaly_detection.ipynb`

Folder:
- /data → berisi dataset
- anomaly_detection.ipynb → notebook utama untuk teori, kode, hasil, dan laporan
